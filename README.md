# 미니 야구 ⚾

탭 한 번으로 즐기는 미니멀 야구 게임. **타자 모드**와 **투수 모드** 두 가지를 제공하며, 각 모드는 9구로 진행됩니다.

- **타자 모드** — 투수 시점의 몰입형 구장 화면. 매 투구마다 3×3 존 중 **랜덤한 칸**으로 공이 점점 커지며 날아오고(목표 칸은 노란 링으로 표시), 그 **칸을 도착 타이밍에 맞춰 터치**해 친다. 정확한 칸 + 정확한 타이밍이 맞아야 장타가 나온다(인접 칸/빗맞은 타이밍은 약한 타구·헛스윙). 상단 HUD(SCORE·BEST·PITCH·OUT), 좌측 MISSION(안타 3개), 우측 SWING 타이밍 피드백, 홈플레이트 버튼으로 구성. 가로·세로 화면 모두 대응.
- **투수 모드** — 마운드 시점의 몰입형 화면. **구종(직구·슬라이더·커브·체인지업)** 을 골라 3×3 존 중 던질 칸을 선택한다. 가운데는 맞기 쉽고 가장자리는 삼진·헛스윙을 노린다. 구종마다 구속·제구·결과 확률이 다르다. 삼진·아웃은 점수 획득, 피안타·실점은 감점 — **점수(SCORE)** 로 최고 기록에 도전하며, MISSION(삼진 3개)도 있다.
- **이닝제 모드** — 타자·투수와 같은 몰입형 구장 화면. 3회 동안 타격하며, 공이 올 때 타이밍 맞춰 화면을 탭해 친다. 안타로 주자를 진루시켜 홈으로 불러들이면 득점, 3아웃이면 다음 회로 교대. 좌측 **주자 베이스 다이아몬드**에 주자가 (팀 색으로) 점등되고, 한 번 시작하면 3회까지 자동 진행된다.

## 기능
- 🎬 **타이틀(표지) 화면** — 야간 구장 배경, 메탈릭 "MINI BASEBALL" 로고, TAP TO START, 최고 점수(BEST SCORE) 표시.
- ⚾ **응원팀 선택** — 표지에서 시작하면 KBO 10개 구단(LG·두산·KT·SSG·NC·KIA·삼성·롯데·한화·키움) 중 응원팀을 고른다(팀 색상 배지). 선택은 저장되어 홈에 표시되고, 홈의 배지로 언제든 변경 가능. **선택한 구단의 대표 색이 플레이 화면(HUD·패널·버튼·스코어보드·투수 유니폼 등)의 포인트 컬러로 반영**된다 (CSS `--pt` 변수 + `color-mix`).
- 🖐️ **손동작 조작** — 웹캠으로 ✊ 스윙 / ✋ 다음 공(타자), ☝️ 조준 + ✊ 투구(투수). 카메라는 직접 켤 때만 작동. (MediaPipe Tasks Vision)
- 🏆 **최고 기록** — 모드별 최고 성적을 브라우저에 저장하고 홈에서 표시.
- 🔥 **콤보** — 연속 안타(타자) / 연속 아웃(투수)를 배지로 표시. 3연타부터 타점 보너스.
- ⚙️ **난이도** — 쉬움 / 보통 / 어려움(공 속도 변화).
- 🔊 **사운드 & 진동** — Web Audio 효과음(외부 파일 없음), 우상단 버튼으로 음소거. 타격 시 화면 흔들림·진동 피드백.

## 디자인
화이트 배경 + 포인트 컬러 블루(`#0017ac`)의 미니멀 스타일. 모바일 세로 화면(세이프에리어 포함)에 맞춰 반응형으로 구성.

## 기술
순수 **HTML / CSS / JavaScript**, 외부 의존성 없음.

## 로컬 실행
`index.html`을 브라우저에서 열기만 하면 됩니다.

## GitHub Pages 배포
1. 이 폴더를 GitHub 저장소에 푸시
2. **Settings → Pages → Branch** 를 `main` / `/ (root)` 으로 설정
3. 발행된 URL 접속

빌드 과정이 없는 정적 사이트라 별도 설정이 필요 없습니다.

## 파일 구성
```
index.html   화면 구조 (홈 / 타자 / 투수 / 결과)
style.css    스타일·애니메이션
script.js    게임 로직 (사운드·콤보·기록·난이도)
gesture.js   손동작 인식 (MediaPipe, ES 모듈)
```

> 손동작 기능은 ES 모듈 + CDN을 사용하므로 `file://` 직접 열기보다는 간단한 로컬 서버에서 실행하길 권장합니다. (예: `npx serve`)
